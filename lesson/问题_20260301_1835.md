# 问题标题
2026-03-01 18:35，燃油页最近记录头像尺寸未显式约束

# 问题描述
燃油页最近记录的头像通过 `leadingBuilder` 直接返回 `DeviceAvatar`，没有显式尺寸约束，导致它使用组件默认半径，和计时页最近记录的头像视觉不一致。

# 根本原因分析
计时页在记录行里对头像使用了外层尺寸约束和显式半径，而燃油页没有，头像尺寸退回到 `DeviceAvatar` 默认值。

# 解决方案与步骤
1. 在燃油页 `leadingBuilder` 中给头像增加固定 `SizedBox`。
2. 将燃油页记录头像尺寸显式设为 `45 x 45`。
3. 同步给 `DeviceAvatar` 和空态 `CircleAvatar` 传入对应半径 `22.5`。

# 改进启示
共用头像组件不等于共用尺寸策略。列表行中的头像尺寸必须在调用层明确约束，否则会因为默认值造成跨页面视觉漂移。
