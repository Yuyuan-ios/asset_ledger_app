# 问题标题
2026-02-28 20:53，项目详情弹窗收款记录左边界与单价区未对齐

# 问题描述
项目详情弹窗中，收款记录的日期金额文本起点比上方设备单价区的设备名起点更靠左，视觉上缺少统一的对齐线。

# 根本原因分析
收款记录列表仍沿用内容区默认左边界，而设备单价区已经形成了“标签列后开始内容”的对齐结构，两者没有共享同一条内容起始线。

# 解决方案与步骤
1. 复用单价区的标签列宽度和列间距。
2. 将收款记录左侧文本区整体右移到与设备名同一条起始线。
3. 保持右侧编辑/删除按钮位置不变。

# 改进启示
同一弹窗内多块列表信息应尽量共享统一的内容起始线，这样扫描路径更稳定，界面也更整齐。
