# 问题标题
2026-02-28 19:56，账户页总览饼图分割线视觉宽度不一致

# 问题描述
账户页总览卡片中的红蓝饼图使用扇区角度留缝作为分隔方式，导致分割缝从圆心到外圈视觉上不呈现等宽线效果。

# 根本原因分析
基于扇区角度裁切的“留缝”本质是楔形空隙，越靠外圈视觉越宽，无法形成像设计稿那样稳定、笔直的等宽分割线。

# 解决方案与步骤
1. 取消饼图扇区之间的角度留缝绘制。
2. 改为先完整绘制两个扇区。
3. 叠加固定线宽的径向分割线，保证从圆心到外圈视觉宽度一致。
4. 将分割方式从角度 token 改为线宽 token，便于后续微调。

# 改进启示
当设计目标是“等宽分割线”时，应使用固定描边表达边界，而不是依赖几何留缝来模拟线条。
