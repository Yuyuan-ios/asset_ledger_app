# 问题标题
2026-02-28 14:05，账户总览卡片层次感不足，页面主信息区不够突出

# 问题描述
账户页的总览卡片与背景区分度偏弱，虽然有边框，但视觉上仍显得偏平，缺少主信息区域应有的轻微悬浮感。

# 根本原因分析
当前总览卡片仅依赖边框和底色与页面背景做区分，没有提供阴影层次。对于承载页面核心汇总信息的卡片，仅靠细边框不足以形成明显视觉重心。

# 解决方案与步骤
1. 仅为账户总览卡片增加一层克制的阴影。
2. 保持现有圆角、边框和布局不变，避免影响整体风格。
3. 将阴影模糊、Y 偏移和透明度收口到 `AccountTokens`，便于后续统一微调。

# 改进启示
主卡片的视觉强调不一定依赖更重的颜色或更粗的边框。轻微、低透明度的阴影往往更适合在不破坏简洁风格的前提下建立信息层级。
