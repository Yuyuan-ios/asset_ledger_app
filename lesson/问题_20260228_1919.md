# 问题标题
2026-02-28 19:19，项目详情弹窗设备单价区左边距需要可调

# 问题描述
项目详情弹窗中的“设备单价”标题和下方设备名称当前直接贴合内容区左边缘，后续需要按视觉稿单独微调左侧距离。

# 根本原因分析
这两处左边距没有进入 token 层，布局只能通过组件代码硬改，不利于快速细调和保持结构清晰。

# 解决方案与步骤
1. 在 `AccountTokens` 中新增项目详情弹窗的左边距 token。
2. 给“设备单价”标题行增加左侧 `Padding`。
3. 给设备名称文本增加左侧 `Padding`。

# 改进启示
弹窗内需要反复微调的对齐参数应尽早落到 token，避免每次都回到组件里改局部布局。
