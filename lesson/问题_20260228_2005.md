# 问题标题
2026-02-28 20:05，已实收绿色偏浅

# 问题描述
账户页项目进度条与总览饼图共用的“已实收”绿色偏浅，视觉存在轻飘感，用户希望颜色更深一些。

# 根本原因分析
`TimingColors.chartIncome` 使用了较高亮度的浅绿色，虽然清爽，但在白底和浅灰底上对比度偏弱。

# 解决方案与步骤
1. 保持统一颜色来源不变。
2. 将 `TimingColors.chartIncome` 从浅绿色调整为更深一档的中绿色。
3. 让所有复用该 token 的进度条与饼图同步更新。

# 改进启示
共享语义色应优先通过统一 token 微调，避免组件各自覆写，保持全局一致性。
