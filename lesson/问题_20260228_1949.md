# 问题标题
2026-02-28 19:49，账户筛选弹窗清空按钮位置偏左

# 问题描述
账户筛选弹窗中，输入框下方的“清空”按钮位于左侧，与当前页面操作重心不一致，用户希望将其调整到右侧。

# 根本原因分析
该按钮使用了左对齐 `Align`，导致视觉上与输入框操作区、底部确认按钮的方向不一致。

# 解决方案与步骤
1. 将“清空”按钮的 `Align` 从 `Alignment.centerLeft` 改为 `Alignment.centerRight`。
2. 保持按钮本身样式和清空逻辑不变，只调整布局方向。

# 改进启示
次级操作按钮也应遵循当前区域的视觉节奏，避免主次动作在水平方向上相互打架。
