# 问题标题
2026-02-28 18:18，账户页总览卡片标题字号位置

# 问题描述
账户页总览卡片中的“总览”标题字号使用硬编码，无法通过 token 统一调整。

# 根本原因分析
标题组件直接写死了 `fontSize: 16`，只把字间距抽到了 `AccountTokens`，导致字号没有进入可维护的设计参数层。

# 解决方案与步骤
1. 在 `lib/tokens/mapper/account_tokens.dart` 中新增 `overviewTitleFontSize`。
2. 将 `lib/patterns/account/account_overview_card_pattern.dart` 中“总览”标题的 `fontSize` 改为引用该 token。

# 改进启示
同一视觉单元的核心尺寸应成组落在 token 中，避免只抽一半参数，后续调整仍要回到组件里找硬编码。
