# 问题标题
2026-02-28 12:23，计时弹窗在键盘贴边后底部表单内容被操作栏遮挡

# 问题描述
将底部操作栏改为键盘弹起时贴近系统键盘后，燃油弹窗表现正常，但计时弹窗由于内容更长，在中文全键盘场景下底部字段与操作栏发生视觉重叠，影响输入体验。

# 根本原因分析
计时弹窗的可滚动内容没有为底部操作栏预留足够的滚动缓冲区。键盘弹起后，操作栏固定在底部，内容区高度变小，最后几项字段仍然会滚到操作栏下方，形成遮挡。

# 解决方案与步骤
1. 仅调整计时弹窗的滚动内容，不改已正常的燃油弹窗。
2. 在计时弹窗的 `SingleChildScrollView` 内容底部增加额外内边距。
3. 这个内边距覆盖操作栏高度、顶部间距和提示区间距，确保末尾字段可以滚到操作栏上方。

# 改进启示
当底部操作栏固定贴边时，长表单不能只依赖 `Expanded + ScrollView`。必须显式为滚动内容预留底部缓冲，否则在软键盘场景下很容易出现最后几项被遮挡的问题。
