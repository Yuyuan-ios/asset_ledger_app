# 问题标题
2026-02-28 21:10，项目详情单价区需改为逐行表格对齐

# 问题描述
项目详情弹窗中，“设备单价”和第一台设备仍存在视觉上的高低差。用户要求按草图改成严格的逐行布局，让左侧标签与右侧设备信息一行一行对应。

# 根本原因分析
此前的实现是“左侧标签列 + 右侧设备列表列”，虽然列宽固定，但左侧是一个独立 `Column`，右侧是另一组多行 `Column`，两边只是整体并排，未形成真正的逐行绑定关系。

# 解决方案与步骤
1. 将单价区重构为逐行表格结构。
2. 第一行固定为“设备单价 + 第一台设备”。
3. 第二行在多设备时固定为“批量修改 + 第二台设备”。
4. 其余设备行左侧留空，但继续沿用同一标签列宽。
5. 将“批量修改”和“修改”按钮切为紧凑样式，避免默认按钮高度破坏行内对齐。

# 改进启示
需要严格同行时，必须让左右内容在同一行容器里一一对应，而不是分别成列后再并排拼接。
