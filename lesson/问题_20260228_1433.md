# 问题标题
2026-02-28 14:33，底部 toast 与屏幕底边距离偏大，贴底感不足

# 问题描述
在 toast 背景已经变轻之后，提示框与页面底边之间的距离仍略大，视觉上有点悬得过高，不够贴近底部轻提示的预期位置。

# 根本原因分析
统一 toast 使用了较保守的底部外边距，虽然安全，但会让轻提示显得离交互区域偏远，削弱“就近反馈”的感觉。

# 解决方案与步骤
1. 只调整 toast 底部 margin，不改变宽度、自适应逻辑和文案样式。
2. 将底部外边距从较大的值收紧到更贴底的位置。
3. 继续保留左右 margin，确保小提示不会贴边。

# 改进启示
轻提示的位置也在传达交互语气。稍微贴近底部，会更像“轻轻提醒”；离底部过远，则更像插入式信息层，容易显得多余。
