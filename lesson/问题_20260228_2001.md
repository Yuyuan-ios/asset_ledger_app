# 问题标题
2026-02-28 20:01，账户页饼图已实收颜色与进度条不一致

# 问题描述
账户页总览卡片中饼图的已实收部分仍使用蓝色，而项目卡片中的收款进度条使用绿色，视觉语言不统一。

# 根本原因分析
饼图和进度条分别使用了不同的颜色来源，导致同一业务语义“已实收”没有共享同一颜色定义。

# 解决方案与步骤
1. 在 `account_tokens.dart` 中引入 `core_tokens.dart`。
2. 将 `overviewPieReceived` 改为复用 `TimingColors.chartIncome`，与项目卡片进度条颜色保持一致。

# 改进启示
相同业务语义应尽量共享同一色值来源，避免同义状态在不同组件里出现多套颜色语言。
