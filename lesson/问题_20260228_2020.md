# 问题标题
2026-02-28 20:20，项目详情弹窗设备单价区信息断层

# 问题描述
项目详情弹窗中“设备单价”标题与设备编号/价格分成上下两块，视觉上有断层。用户希望设备编号和价格直接接在“设备单价”后面，多台设备时继续在第一台设备编号下方对齐。

# 根本原因分析
现有布局把标题行和设备列表拆成两个独立区块，导致左侧标签与右侧内容之间存在不必要的垂直断开。

# 解决方案与步骤
1. 将“设备单价”改为左侧固定标签列。
2. 将设备编号、价格、修改按钮放入右侧内容列，并与标签同行起始。
3. 多设备时，后续设备继续在第一条设备信息下方垂直对齐。
4. 多设备场景保留“批量修改”，放到该区域右上角。

# 改进启示
表单中的标签与其对应内容应尽量形成稳定的行内结构，避免被拆成两段造成阅读跳跃。
