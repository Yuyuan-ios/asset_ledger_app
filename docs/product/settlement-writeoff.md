# 结清/核销规则

## 核心规则

- 同一项目唯一核销记录。
- 结清表示将当前 remaining 全额核销。
- 撤销结清或核销时直接恢复对应项目的 remaining。

## 行为边界

- 不为同一项目重复创建多条并列核销记录。
- 结清与撤销应保持可逆，不能留下重复抵扣。
- 展示、统计、账户页应读取同一套核销结果。
